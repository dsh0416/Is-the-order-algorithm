**C++ (2-1.cpp):**

```
#include <iostream>
using namespace std;
int main(){
	cout<<"Hello World";
	return 0;
}
```

**Python (2-1.py):**

```
print("Hello World")
```

运行后如果屏幕上出现了一行 Hello World 就代表你的测试环境调通了，可以开始接下来的工作了。不过在此之前，我们先来解释一下之前写的代码是什么意思。

**C++:**

```
#include <iostream> //引用输入输出库，允许程序从键盘读取内容和在屏幕输出文字
using namespace std;//使用标准命名空间，常见的许多代码都是在这命名空间下先声明一下
int main(){ //定义这是主程序，C++中必须要有主程序，这是程序运行后第一个运行的程序
	cout<<"Hello World";//输出Hello World，防止和代码混淆，字符串用英文引号引起来
	return 0;//正常退出程序，程序运行完毕
}//与之前的定义时的左大括号对齐，表示定义完毕
```

**Python:**

```
print("Hello World")
# 输出 Hello World，防止和代码混淆，字符串用英文引号引起来
```

`TIP:`程序的编译和运行通常是非常严格的，尤其是括号、引号、分号的使用，C++还要注意是<<还是>>，请务必注意我给出样例代码中这些符号的书写，并且这些符号都是英文输入法下的，请务必不要用中文输入法输入这些符号。如果你是初学程序，建议学习过程中多敲几遍代码，并可以调整一些细节试试，以让自己慢慢熟悉每一行代码的意义。而 Python 要特别注意缩进，因为 Python 的缩进直接代表了程序的逻辑关系。

掌握了上面的语法你就可以进行输出任何文字了，但是你还需要掌握输入，这是显然的，比如你想写一个加法的计算器，你肯定是希望每次运行时只要输两个数字进去，算出来就行。而不是每次都去改动代码，重新编译运行。所以你的程序还需要有输入的功能。下面再给出两个样例。

**C++ (2-2.cpp):**

```
#include <iostream>
using namespace std;
int main(){
	int number;
	cin>>number;
	cout<<number;
	return 0;
}
```

**Python (2-2.py):**

```
number = input()
print(number)
```
运行后，运行界面会一直闪烁光标，直到你输入一个整数并按回车后，程序会输出你输入的整数。如果没有问题，我来解释一下这个代码。

```
#include <iostream>
using namespace std;
int main(){
	int number; //定义 number 是一个整数变量
	cin>>number;//将键盘输入的一个整数赋值给number
	cout<<number;//输出number的值
	return 0;
}
```

**Python:**

```
number = input()
# 定义变量 number 为键盘输入的一个字符串
#（需要注意是这里的输入是一个字符串，如果要运算需要转换）
print(number)
# 输出number的值
```

上面 number 这个变量是我随便定义的，你们也可以给它起不同的名字。只要调用和定义的变量类型一致即可。需要特别注意的是 C++ 的 int 类型只能存储一个 -2147483648 至 2147483647 的整数，更崩溃的是如果你运算过程中超过了它的上限，它会直接跳到它的下限上而不是报错，所以需要在编写代码的过程中尤其当心和注意。而 Python 则没有这样的限制，想多大就多大，想是整数还是小数随便你。

为了帮助使用 C++ 用户理解其中的变量类型，这里特别列出常用的 C++ 基本变量类型

```
bool 布尔型，只能存储 true 还是 false 这两个值
int 整型 -2147483648 至 2147483647 的整数
unsigned int 无符号整型，只有正数的整型，范围 0 至 4294967295
long 长整型，以前是这样，现在和 int 没有区别
long long 长长整型，存储的范围是 –9223372036854775808 至 9223372036854775807
unsigned long long 无符号长长整型 0 至 18446744073709551615
float 浮点数，可存储很大的数和小数（精度堪忧）
double 双精度浮点数，可存储很大的数和小数（精度是float两倍）
```

之所以会有这样的限制是因为 C++ 采用固定的内存大小来存储变量。所以需要在声明的时候要规定好大小。如果你对此感到有兴趣，机智的你其实可以发现上面这些看起来很奇怪的范围的数字其实都和2有些关系，都是2的某个次方数，或是2的某个次方数减1，因为内存的本质只能存储1或0这两种状态，用多个这样的存储位来存储，所以数据范围都和2的某个次方数有关。

### 条件判断

条件判断是一个非常重要而实用的特性，这规定了程序在某些条件下执行某些命令，也就具备了基本的决策能力。非常有意思的是大多数语言的条件判断命令是非常接近的，都是 if。什么是 if ？就是“如果”，再简化的话，就是一个字“当”。这么一说很多同学立刻就理解了，“当” 不就是：

```
当山峰没有棱角的时候
当河水不再流 
当时间停住日夜不分
当天地万物化为虚有 
我还是不能和你分手
不能和你分手 
你的温柔是我今生最大的守候 
```

有书里曾说这是一个典型的程序上的四重判断，用 Python 的伪代码表示一下就是：

```
if 山峰没有棱角:
	if 河水不流:
		if 时间停住日夜不分:
			if 天地万物化为虚有:
				我还是不能和你分手
				不能和你分手 
				你的温柔是我今生最大的守候 
```

用 C++ 的伪代码就是

```
if(山峰没有棱角){
	if(河水不流){
		if(时间停住日夜不分){
			if(天地万物化为虚有){
				我还是不能和你分手;
				不能和你分手;
				你的温柔是我今生最大的守候;
			}
		}
	}
}
```
理解起来就是说判断是不是满足山峰没有了棱角，假如真的没棱角了，再判断河水是不是在流，假如河水真的不再流了，就再判断时间是否停止，假如时间真的停止了再判断天地万物是不是化为了虚有，假如也化为了虚有，以上这些判断都是真的，没有一个是假的的时候就开始执行三条语句`我还是不能和你分手`，`不能和你分手`，`你的温柔是我今生最大的守候`。

事实上... 这样理解是错误的。因为，歌曲根本表达的不是这个意思。歌曲想表达的并不是山峰没有棱角同时河水不流同时时间停住同时天地万物化为虚有的时候才不分手，而是无论上面哪种成立都不分手。所以根本不需要同时满足这四个条件，根本不是四重判断，满足任何之一都不应该分手。

所以修正后的 Python 程序应该是：

```
if 山峰没有棱角 or 河水不流 or 时间停住日夜不分 or 天地万物化为虚有:
	我还是不能和你分手
	不能和你分手 
	你的温柔是我今生最大的守候 
```

C++ 程序是

```
if(山峰没有棱角 || 河水不流 || 时间停住日夜不分 || 天地万物化为虚有){
	我还是不能和你分手;
	不能和你分手;
	你的温柔是我今生最大的守候;
}
```

这么理解看起来正确了不少，但是其实还是错的。因为，难道排除掉这四种世界级灾难的时候就可以分手了？显然不是嘛！作者的意思是，无论是什么情况，都不应该分手，所以正确的 Python 程序是：

```
if True:
	我还是不能和你分手
	不能和你分手
	你的温柔是我今生最大的守候
```

正确的 C++ 程序是：

```
if(true){
	我还是不能和你分手;
	不能和你分手;
	你的温柔是我今生最大的守候;
}
```

其实说到底，根本就没有判断啊！里面这三条命令在无论什么条件下都会执行啊喂！写出这么冗余的代码也就只有琼瑶阿姨了啊！诶，其实说那么多我是想给大家来介绍一下条件判断的常见形式和使用方法。

其实条件判断的实质是判断一个表达式是真还是假，如果是真就会执行。（假会执行else语句，当然else可以不要）。执行的内容也可以嵌套另一个if。至于里面是什么表达式，随便你构造。if 的语法如下

**C++:**

```
if(表达式){
	执行语句
}else if (表达式2){
	执行语句
}else if (表达式3){
	执行语句
}else{
	执行语句
}
```

**Python:**

```
if 表达式:
	执行语句
elif 表达式2:
	执行语句
elif 表达式3:
	执行语句
else:
	执行语句
```

表达式中可以使用 >表示大于、<表示小于、==表示等于（注意是两个=号，单个=表示赋值不代表比较）、!=表示不等于。表达式中可以使用几者同时成立（Python 中为 and，C++ 中为 &&），或者几者只要有一个成立都为成立（Python 中为 or，C++ 中为 ||）来简化判断的表达式。下面给出一道样例题目，快去试试吧。

输入一个数字，输出它的绝对值。

**C++:**

```
#include <iostream>
using namespace std;
int main(){
	int num;
	cin>>num;
	if(num<0){
		cout<< -num;
	}else{
		cout<< num;
	}
	return 0;
}
```

**Python:**

```
num = input()
num = int(num)
if num<0:
	print(-num)
else:
	print(num)
```

输入一个整数，如果大于80输出Excellent，大于60但不大于80时输出Great，大于80时输出Excellent，小于60时输出Good。

**C++:**

```
#include <iostream>
using namespace std;
int main(){
	int num;
	cin>>num;
	if (num>80){
		cout<<"Excellent";
	}else if(num>60){
		cout<<"Great";
	}else{
		cout<<"Good";
	}
}
```

**Python:**

```
num = input()
num = int(num)
if num>80:
	print("Excellent")
elif num>60:
	print("Great")
else:
	print("Good")
```

输入一个整数，如果是小于0的，输出“-1”，而如果输入的大于0，输出数字乘以2的值。

**C++:**

```
#include <iostream>
using namespace std;
int main(){
	int num;
	cin>>num;
	if (num<0){
		cout<<"-1";
	}else{
		cout<<num*2;
	}
}
```

**Python:**

```
num = input()
num = int(num)
if num<0:
```
